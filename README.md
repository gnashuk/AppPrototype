Compatibility
-------------
Requires iOS 11.0 or above

Installation
-----------
### Prerequisites
 - XCode 9
 - CocoaPods
 
### Installation Steps
 1. Generate XCode workspace through CocoaPods:
 ```shell
pod install
```
 2. Open `AppPrototype.xcworkspace`
